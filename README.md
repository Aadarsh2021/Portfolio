About
This repository contains the code for my portfolio website. The website is built using HTML, CSS, and JavaScript. It showcases my skills and experience in web development, machine learning, and data analysis.

Features
The website includes the following features:

A home page with information about my background, skills, and work experience
A portfolio page with examples of my work
A contact page with ways to get in touch with me
Usage
To view the website, simply clone this repository and open the index.html file in a web browser.
